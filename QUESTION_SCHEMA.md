# Структура данных вопроса (Question)

## Обязательные поля

| Поле         | Тип                                              | Описание                         |
|--------------|--------------------------------------------------|----------------------------------|
| `id`         | `string`                                         | Уникальный идентификатор вопроса |
| `category`   | `string`                                         | Категория вопроса (произвольная строка) |
| `type`       | `"text"` \| `"audio"` \| `"video"` \| `"image"` \| `"creative"` \| `"together"` | Тип вопроса |
| `difficulty` | `"easy"` \| `"medium"` \| `"hard"`              | Сложность вопроса                |
| `points`     | `number`                                         | Количество очков за ответ        |
| `question`   | `string`                                         | Текст вопроса (поддерживает `\n` для переносов) |
| `answer`     | `string`                                         | Текст правильного ответа         |

---

## Необязательные поля

| Поле              | Тип                    | Описание                                                                 |
|-------------------|------------------------|--------------------------------------------------------------------------|
| `mediaPath`       | `string`               | Путь к медиафайлу относительно `/public/media/`. Используется при `type: "audio"`, `"video"`, `"image"` |
| `questionImages`  | `string` \| `string[]` | Одно или несколько изображений, показываемых **вместе с вопросом**. Путь относительно `/public/media/` |
| `answerImages`    | `string` \| `string[]` | Одно или несколько изображений, показываемых **вместе с ответом**. Путь относительно `/public/media/` |
| `answerVideoPath` | `string`               | Путь к видео, которое показывается после раскрытия ответа. Путь относительно `/public/media/` |
| `thinkingTime`    | `number`               | Индивидуальный таймер для этого вопроса (в секундах). Если не задан, используется значение из настроек игры |

---

## Типы вопросов (`type`)

| Значение    | Описание                                                             |
|-------------|----------------------------------------------------------------------|
| `text`      | Обычный текстовый вопрос                                             |
| `audio`     | Вопрос с аудиофайлом — нужно заполнить `mediaPath` (`.mp3`, `.wav` и т.д.) |
| `video`     | Вопрос с видеофайлом — нужно заполнить `mediaPath` (`.mp4` и т.д.)  |
| `image`     | Вопрос с изображением — нужно заполнить `mediaPath` (`.jpg`, `.png` и т.д.) |
| `creative`  | Творческое задание — команда выступает, ведущий оценивает            |
| `together`  | Совместное задание — все команды отвечают вместе и получают очки     |
| `ordering`  | Задание на упорядочивание — нужно расставить элементы в правильном порядке |

---

## Сложность (`difficulty`)

| Значение | Отображение | Цвет     |
|----------|-------------|----------|
| `easy`   | Легкий      | Зелёный  |
| `medium` | Средний     | Жёлтый   |
| `hard`   | Сложный     | Красный  |

---

## Примеры

### Текстовый вопрос
```json
{
  "id": "q1",
  "category": "История",
  "type": "text",
  "difficulty": "easy",
  "points": 1,
  "question": "Как называется столица России?",
  "answer": "Москва"
}
```

### Аудио-вопрос
```json
{
  "id": "q2",
  "category": "Музыка",
  "type": "audio",
  "difficulty": "medium",
  "points": 2,
  "question": "Угадай мелодию",
  "answer": "Название песни",
  "mediaPath": "folder/song.mp3"
}
```

### Вопрос с фото к вопросу и ответу
```json
{
  "id": "q3",
  "category": "Искусство",
  "type": "image",
  "difficulty": "hard",
  "points": 3,
  "question": "Кто написал эту картину?",
  "answer": "Леонардо да Винчи",
  "mediaPath": "art/mona_lisa.jpg",
  "questionImages": ["art/detail1.jpg", "art/detail2.jpg"],
  "answerImages": ["art/answer.jpg"]
}
```

### Совместное задание с видео-ответом
```json
{
  "id": "q4",
  "category": "Кино",
  "type": "together",
  "difficulty": "medium",
  "points": 2,
  "question": "Назовите фильм по описанию...",
  "answer": "Название фильма",
  "answerVideoPath": "movies/clip.mp4"
}
```

### Творческое задание с таймером
```json
{
  "id": "q5",
  "category": "Творчество",
  "type": "creative",
  "difficulty": "medium",
  "points": 2,
  "thinkingTime": 60,
  "question": "Покажи жестами песню — команда угадывает.",
  "answer": "Засчитывается правильное угадывание командой."
}
```

### Задание на упорядочивание
```json
{
  "id": "q6",
  "category": "История",
  "type": "ordering",
  "difficulty": "hard",
  "points": 3,
  "question": "Расположите события в хронологическом порядке:\nСобытие А, Событие Б, Событие В, Событие Г.",
  "answer": "1. Событие В;\n 2. Событие А;\n 3. Событие Г;\n 4. Событие Б."
}
```

## Расположение медиафайлов

Все пути в полях `mediaPath`, `questionImages`, `answerImages`, `answerVideoPath` указываются **относительно** папки `/public/media/`.

```
/public/media/
  newyear/        ← папка для конкретного игрового набора
    photo1.jpg
    audio.mp3
  mr210226/
    h2.jpeg
    Ближе_быть.mp3
```

Пример: если файл лежит в `/public/media/newyear/photo1.jpg`, то в JSON пишем `"newyear/photo1.jpg"`.
