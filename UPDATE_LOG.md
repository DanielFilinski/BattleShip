# Обновление игры "Морской бой" - Декабрь 2025

## Что изменено

### 1. Добавлены 4 режима игры

- **Хор** (choir) - Музыкальные вопросы и задания
- **Библейские знания** (biblical) - Вопросы о Библии и христианской вере
- **Эрудиция** (erudition) - Общие знания: наука, история, литература, география, искусство
- **Общие знания** (general) - Природа, спорт, еда, технологии, развлечения

### 2. Структура вопросов

Каждый режим содержит **25 вопросов**:
- **20 вопросов** для кораблей (различной сложности: 1-3 балла)
- **5 вопросов-бомб** (повышенной сложности: 3 балла)

### 3. Уникальные расположения

- Каждый режим имеет **уникальное расположение кораблей**
- Каждый режим имеет **уникальное расположение бомб**
- Бомбы содержат интерактивные задания для команд

### 4. Файловая структура

#### Файлы вопросов:
- `public/data/questions.json` - Режим "Хор" (25 вопросов)
- `public/data/questions-biblical.json` - Режим "Библейские знания" (25 вопросов)
- `public/data/questions-erudition.json` - Режим "Эрудиция" (25 вопросов)
- `public/data/questions-general.json` - Режим "Общие знания" (25 вопросов)

#### Файлы кораблей:
- `public/data/ships.json` - Режим "Хор"
- `public/data/ships-biblical.json` - Режим "Библейские знания"
- `public/data/ships-erudition.json` - Режим "Эрудиция"
- `public/data/ships-general.json` - Режим "Общие знания"

#### Файлы бомб:
- `public/data/bombs.json` - Режим "Хор"
- `public/data/bombs-biblical.json` - Режим "Библейские знания"
- `public/data/bombs-erudition.json` - Режим "Эрудиция"
- `public/data/bombs-general.json` - Режим "Общие знания"

#### Конфигурация режимов:
- `public/data/gameModes.json` - Описание всех режимов игры

### 5. Интерфейс выбора режима

На стартовом экране добавлен интерфейс выбора режима:
- Визуальные карточки с описанием каждого режима
- Цветовая индикация выбранного режима
- Автоматическая загрузка соответствующих вопросов и расположений

### 6. Документация

Создан файл `SHIP_POSITIONS_AND_QUESTIONS.md` с полной документацией:
- Расположение всех кораблей для каждого режима
- Список всех вопросов с ответами
- Расположение бомб для каждого режима
- Карта игрового поля

## Как играть

1. Откройте игру
2. Выберите режим игры на стартовом экране
3. Введите названия команд
4. Нажмите "Начать игру"
5. Игра автоматически загрузит вопросы и расположения для выбранного режима

## Технические изменения

### Обновленные компоненты:
- `src/App.tsx` - Поддержка выбора режима и динамической загрузки данных
- `src/components/WelcomeScreen.tsx` - Интерфейс выбора режима
- `src/utils/loadData.ts` - Функции загрузки данных в зависимости от режима
- `src/types/game.ts` - Новый тип `GameMode`

### Особенности реализации:
- Динамическая загрузка данных на основе выбранного режима
- Сохранение совместимости с существующим кодом
- Расширяемая архитектура для добавления новых режимов

## Добавление новых режимов

Для добавления нового режима:
1. Создайте файлы `questions-[mode].json`, `ships-[mode].json`, `bombs-[mode].json`
2. Добавьте описание режима в `gameModes.json`
3. Режим автоматически появится в интерфейсе
