# Видео файлы для вопросов

Поместите видео файлы для вопросов в эту папку.

## Формат
- Формат: MP4, WebM
- Название файлов должно совпадать с тем, что указано в `public/data/questions.json` в поле `mediaPath`

## Пример

В questions.json:
```json
{
  "id": "q31",
  "type": "video",
  "mediaPath": "video/movie_clip.mp4"
}
```

Файл должен находиться: `public/media/video/movie_clip.mp4`
