# Звуковые файлы

Поместите следующие звуковые файлы в эту папку:

- **hit.mp3** - звук попадания в корабль
- **miss.mp3** - звук промаха
- **correct.mp3** - звук правильного ответа
- **wrong.mp3** - звук неправильного ответа

Все файлы должны быть в формате MP3.

## Где взять звуки

Вы можете:
1. Использовать бесплатные звуки с сайтов типа freesound.org
2. Записать свои звуки
3. Использовать звуки из игр

## Примеры:
- hit.mp3: звук взрыва, удара
- miss.mp3: звук всплеска воды
- correct.mp3: звук колокольчика, аплодисментов
- wrong.mp3: звук гудка, неудачи
